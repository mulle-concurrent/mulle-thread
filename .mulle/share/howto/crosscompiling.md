# mulle-sde Crosscompiling
<!-- Keywords: crosscompile, windows, linux -->

The `mulle-sde platform` is used to activate and deactivate cross-platform
developent. See its help file for info.

You will need a toolchain file in `cmake/` or `cmake/shared` to see support
for a specific platform. Currently only crosscompiling to windows is
possible with  **mulle-clang-project-windows**
